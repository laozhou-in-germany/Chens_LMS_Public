# Use journal

With this tutorial, you'll learn to create a daily note and an event note.  


## Create a daily note

- Show the right sidebar by clicking `Ctrl + Shift + R`
- Active the pane `Calendar` 
- Click a date of your choice
- **Done**: A daily note is created.
