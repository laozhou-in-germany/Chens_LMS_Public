# 99_我的自定义快捷键

| 功能                   | 快捷键           | 相关插件   |
| ---------------------- | ---------------- | ---------- |
| 删除当前笔记页         | Ctrl + Shift + D |            |
| 在当前位置加入模板文字 | Alt + E          | Templater  |
| 打开外部编辑器         | Alt + O          |            |
| 固定该页面             | Alt + P          |            |
| 切换专注模式           | Alt + F          | Focus Mode |
| 切换超级专注模式       | Ctrl + Alt +F    | Focus Mode |
| 向下移动列表项         | Shift +Alt+ Down | Outliner   |
| 向上移动列表项         | Shift +Alt+ Up   | Outliner   |
| 展开列表项             | Ctrl + Down      | Outliner   |
| 闭合列表项             | Ctrl + Up        | Outliner   |
