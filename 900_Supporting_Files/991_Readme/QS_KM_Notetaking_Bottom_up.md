# Notetaking bottom-up
There are generally two approaches to taking notes:
- Top-down
- Bottom-up

## 5 Steps
The bottom-up approach is more suitable to generate original insight from daily random ideas. This vault is designed to support this process. 
It has five steps: 
- Distill the literature note
- Link the atomic note
- Garden the evergreen note
- Develop a topic 
- Link topic to the home node


### Distill the literature note
It's suggested to use "progressive summarization" from Tiago to read the literature. it contains the following steps.
1.  Note
2.  Bold passage
3.  Highlighted passage
4.  Mini-summary
5.  Remix

For more detailed information you can check the article [Progressive Summarization: A Practical Technique for Designing Discoverable Notes - Forte Labs](https://fortelabs.co/blog/progressive-summarization-a-practical-technique-for-designing-discoverable-notes/)

In the demo file [QS - berlin (wikipedia)](../../500_Knowledge_Management/510_📔Literature_Notes/QS%20-%20berlin%20(wikipedia).md) you can find the texts, which are bolded and highlighted. There is also a summary of the article.

Meanwhile, You should extract valuable information from the literature note: 

- atomic notes: information that can be reused later. 
- evergreen notes: your idea, original thinking

In the demo file, one atomic note and one evergreen note were extracted: [Ich bin ein Berliner - Kennedy](../../500_Knowledge_Management/530_⚛️Atomic_Notes/Ich%20bin%20ein%20Berliner%20-%20Kennedy.md), [QS - German cities are beautiful](../../500_Knowledge_Management/550_🌲Evergreen_Notes/QS%20-%20German%20cities%20are%20beautiful.md)

### Link the atomic note
The atomic notes don't have a meaning if they are not used later. So try to reference the atomic notes in evergreen notes and topics.
