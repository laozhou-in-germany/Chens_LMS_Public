# 5 ways to create new notes
[Back to README](../../README.md)

Depending on the note type, there are several ways to add new notes

1. Use shortcut **Ctrl + n** or "New Note" Button
2. Use shortcut **Alt + n** [^1]
3. Use buttons on the dashboards or the notes [^1] [^2] [^3]
4. Right-click the note folder [^1]
5. Click on the days or weeks on the Calendar panel [^3]


[^1]: (Templater plugin required)
[^2]: (Buttons plugin required)
[^3]:  (Calendar plugin required)

## Use shortcut **Ctrl + n** or "New Note" Button

- Create an empty new note by using the shortcut **Ctrl + n** or the "New Note" button on the "File explorer" panel. 
- Change the file name
- Insert the template by using the shortcut **Alt+ n** 
- Move the note to the final note folder

## Use shortcut **Alt + n** 

- Create a new note from a template by using the shortcut **Alt + n** 
- Change the file name
- Move the note to the final note folder

## Use buttons on the dashboards
- Open the dashboard notes (e.g. Goals & Projects Dashboard)
- Click on the button
- The new note is created automatically in the related folder
- Change the file name

## Right-click the note folder

- Expand the left sidebar and active the **File explorer**, if not yet
- Navigate to the related folder for the note (e.g. "110_🏛Pillars" for pillar note)
- Right-click the note folder and create a **new note** using the context menu
- Change the file name

## Click on the days or weeks on the Calendar pane

- Expand the right sidebar and active the **calendar**, if not yet
- Click on the day or the week to create a new note
