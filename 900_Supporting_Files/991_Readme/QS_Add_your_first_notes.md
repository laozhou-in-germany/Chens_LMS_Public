# How to add notes

Depending on the note type, there are several ways to add new notes

1. Use shortcut **Ctrl + n** or "New Note" Button
2. Use shortcut **Alt + n** [^1]
3. Use buttons on the dashboards or the notes [^1] [^2] [^3]
4. Right-click the note folder [^1]
5. Click in the days or weeks on the Calendar panel [^3]


[^1]: (Templater plugin required)
[^2]: (Buttons plugin required)
[^3]:  (Calendar plugin required)

## Solution 1

### Create an empty note
You can easily create an empty note in the following two ways. 
1. Click **New note** Button in the **File explorer** 
2. pressing the keyboard shortcut **Ctrl + n** 

### Insert the frontmatter using a template 
> Plugin *Templater* required.
1. Click the shortcut **Alt + t**
2. Select the wished template from the list

The notes are created in the folder **000_Drafts** by default. You can move them later in the final folder. E.g. **510_📔Literature_Notes** manually.  

## Solution 2
### Creating an empty note using a template
> Plugin *Templater* required.

1. Click the shortcut **Alt + n**
2. Select the wished template from the list

The notes are created in the folder **000_Drafts** by default. You can move them later in the final folder. E.g. **510_📔Literature_Notes** manually.  

## Solution 3
> Plugin *Templater* required.

1. Navigate to the folder for the note **510_📔Literature_Notes**
2. Right-click on the folder to show the context menu
3. Select the menu item **New note**
