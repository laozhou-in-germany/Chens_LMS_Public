README

LMS (Life Management System) is a personal knowledge management system and goal & project management system based on Obsidian.md.

The LMS is a complex interactive system. To make it easy to understand, I consider the LMS as a medieval town. This town has 6 main districts. For more explanation about LMS as a medieval town see:

Plugins (integrated)

  • Admonition: Adds admonition block-styled content to Obsidian.md
  • Alx-folder-note: Add description, summary, and more info to folders with folder notes. (github.com)
  • Buttons: Buttons in Obsidian
  • Calendar: Simple calendar widget for Obsidian.
  • Completr: Auto-completion plugin for the obsidian editor.
  • Daily-notes-viewer
  • Dataview: A high-performance data index and query language over Markdown files, for https://obsidian.md/.
  • Filename heading sync: Obsidian.md plugin to keep the filename and the first header of the file in sync
  • Folder-note-core: Provide core features and API for folder notes in obsidian
  • MetaEdit: MetaEdit for Obsidian
  • Periodic-notes: Create/manage your daily, weekly, and monthly notes in Obsidian
  • Obsidian-columns
  • Obsidian-tasks: Task management for the Obsidian knowledge base.
  • Outliner: Work with your lists like in Workflowy or RoamResearch
  • Supercharged links: obsidian plugin to add attributes and context menu options to internal links
  • Templater: A template plugin for obsidian

Plugins (Optional)

  • Excalidraw plugin: A plugin to edit and view Excalidraw drawings in Obsidian
  • Tag-wrangler: Rename, merge, toggle, and search tags from the Obsidian tag pane
